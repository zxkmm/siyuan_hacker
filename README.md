# SiYuan Hacker Theme

This plugin is intended to give you the best experience on KDE Plasma and other Linux desktop environments, making it as close to the original as possible.

You can also modify the window control buttons, outline, and padding with this plugin: [siyuan_main_window_modification](https://github.com/zxkmm/siyuan_main_window_modification)
