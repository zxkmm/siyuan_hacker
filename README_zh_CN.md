# 思源笔记黑客主题 (Hacker Theme)

本插件旨在为您在 KDE Plasma 和其他 Linux 桌面环境上提供最佳体验，使其尽可能接近原版。

您还可以使用此插件修改窗口控制按钮、轮廓和间距：[siyuan_main_window_modification](https://github.com/zxkmm/siyuan_main_window_modification)
